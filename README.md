# rf_postmortem

RF and BPM Postmortem Backend

Source          | <https://gitlab.diamond.ac.uk/controls/ioc/CS/cs-di-ioc-07>
:---:           | :---:
Docker          | `docker run ghcr.io/diamondlightsource/CS-DI-IOC-07:latest`
Releases        | <https://gitlab.diamond.ac.uk/controls/ioc/CS/cs-di-ioc-07/releases>


## Running

Run as follows:

```
python -m rf_postmortem
```

The flag `-D` can be used to run in debug mode.
